Fake Linux Terminal - Online Simulator

This is example of using jQuery Terminal Emulator to create Fake Linux System. It's work in progress. Check GitHub Repo for more information.

You can call it a fake command line website, that emulates real GNU/Linux system.

This is Open Source and Free software.
